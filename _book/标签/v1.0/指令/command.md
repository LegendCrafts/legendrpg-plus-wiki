# 指令

## 指令索引

|指令      |权限节点 |说明          |
| -------- | ------ | ------------ |
|/lr tag help|lr.tag.help|查看当前模块的所有指令|


## 标签指令帮助

|指令      |权限节点 |说明          |
| ------- | -------- | ------ | ------------ |
|/lr tag info <玩家>|lr.tag.info|查看玩家的标签|
|/lr tag set <标签名> <标签值> <玩家>|lr.tag.set|设置标签|
|/lr tag del <标签名> <玩家>|lr.tag.del|删除标签|
