# specialLabel.yml

## 特殊标签配置
```yaml
#特殊标签配置
#这个配置文件中的特殊标签不能自己新增

#耐久值标签
#可选值: #(num)# : 物品当前耐久度 , #(max_num)# : 物品最大耐久度
durability: '§f[耐久度] #(num)# / #(max_num)#'
#主手标示标签
#装备lore中含有该标签,拿到主手上才能享受装备中的属性
mainItem: '[主手]'
#装备lore中含有该标签,拿到副手上才能享受装备中的属性
offItem: '[副手]'
#装备有效期,过期则删除
timeLimit: '§f[期限] #(date)#'
```
