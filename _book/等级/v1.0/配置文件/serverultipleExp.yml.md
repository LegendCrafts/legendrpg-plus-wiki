## 服务器多倍经验配置文件

```  yaml
#当你看到这个配置文件的时候,作者建议不要人为去配置这个文件
#你可以通过指令的方式对"服务器多倍经验"进行设置
#虽然不建议你手动配置该文件,但还是将注释打上了
#
#经验倍值: (就是几倍经验的意思)
multiple: 1.0
#经验的叠加类型
#可取值: 
#   PLUS : 加算
#   RIDE : 乘算
#   NONE : 玩家的多倍经验覆盖服务器的多倍经验
#   COVER : 服务器的多倍经验覆盖玩家的多倍经验
superposeExpType: COVER
#持续时间
sustainTime: 0
#线程任务ID(无需配置)
taskId: 
#开始时间
useTime: 0
```
